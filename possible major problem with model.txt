  Overall Accuracy = 0.772

  77.2% of all 20,335 predictions were correct.

  But accuracy hides how it’s making mistakes. Precision/recall shows that.

  Per-Class Breakdown
  1️⃣ left_turn

  Precision = 0.97

  Recall = 0.73

  F1 = 0.84

  Support = 6396 samples

  What this means:

  Precision 0.97
  When the model predicts left_turn, it is correct 97% of the time.
  → Almost no false positives.

  Recall 0.73
  It only detects 73% of all actual left turns.
  → It misses 27% of real left turns.

  Interpretation:
  It is very conservative about predicting left_turn.
  When it says left_turn, it’s almost always right — but it misses many.

  2️⃣ neutral

  Precision = 0.61

  Recall = 0.99

  F1 = 0.76

  Support = 6968

  What this means:

  Recall 0.99
  It detects almost every real neutral.
  → Very few neutral false negatives.

  Precision 0.61
  But when it predicts neutral, it is wrong 39% of the time.
  → Lots of false positives.

  Interpretation:
  The model over-predicts neutral.
  It is using neutral as a “safe fallback.”

  This is common in gesture systems.

  3️⃣ right_turn

  Precision = 0.99

  Recall = 0.59

  F1 = 0.74

  Support = 6971

  What this means:

  Precision 0.99
  When it predicts right_turn, it’s almost never wrong.

  Recall 0.59
  But it misses 41% of real right turns.

  Interpretation:
  Same behavior as left_turn:
  Extremely confident but under-detecting.

  What Pattern Is Happening?

  Your model behavior:

  Very high precision for left/right

  Very low precision for neutral

  Extremely high recall for neutral

  Lower recall for left/right

  This means:

   The model is biased toward predicting neutral
   Many left/right samples are being misclassified as neutral

  That explains:

  Neutral recall = 0.99

  Neutral precision = 0.61

  Neutral is absorbing mistakes.

  Why F1 Looks Moderate

  Left/right:

  High precision

  Lower recall

  Neutral:

  High recall

  Lower precision

  F1 punishes imbalance between precision and recall.

  So even though some numbers are very high (0.99), F1 stays in the mid-0.7 range.

  What This Means For Your Gesture System

  Your model is:

  Very cautious about predicting turns

  Aggressive about predicting neutral
